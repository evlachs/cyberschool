# Станция ПК1 — endpoint («Точка А»)

## Легенда

Вы — оперативник «Точка А», первое звено в цепи связи. У вас на руках сведения, которые должны
дойти до «Точки Б» на другом конце — через территорию, где, вероятно, работает противник.
Довериться каналу сразу нельзя: сначала нужно убедиться, что связь вообще жива, затем — что оба
рубежа шифрования на пути держатся стабильно. Только после этого вы вправе передать настоящую
секретную информацию.

## Ваша задача

Первыми подайте голос в канал и убедитесь, что связь установлена. Не спешите с главным
сообщением: дождитесь, пока оба шифроузла на вашем пути подтвердят, что уровни защиты работают,
и координируйтесь с «Точкой Б» — только затем отправляйте настоящие секретные сведения.

## Запуск

Запустить программу-станцию:
1. Узнайте IP-адрес вашей машины с помощью сетевой утилиты ipconfig
2. Добавьте IP-адрес в пропущенное поле конфигурации
3. Сохраните изменения в файле
4. Запустите программу-станцию из командной строки: python3 station.py --config configs/endpoint.json

Для связи с Мастером и получения подсказок используйте канал спец. связи через открытое окно
на вашей Станции.

## Команды

```
send <текст>   — отправить сообщение соседу
inbox [n]      — показать последние n полученных сообщений
```
