# Станция ПК7 — endpoint («Точка Б»)

## Легенда

Вы — оперативник «Точка Б», конечная точка цепи связи, куда должны дойти сведения от «Точки А»
через территорию противника. Прежде чем доверять каналу, нужно убедиться, что связь жива, а
затем — что оба рубежа шифрования на пути работают стабильно в обе стороны. Только тогда канал
готов к передаче настоящей секретной информации.

## Ваша задача

Подтверждайте каждый пройденный рубеж защиты и держите канал готовым принять секретные сведения
от «Точки А» — а при необходимости отправить сведения в ответ. Следите за тем, что приходит:
если сообщение нечитаемо, значит рубежи шифрования ещё не согласованы.

## Запуск

Запустить программу-станцию:
1. Узнайте IP-адрес вашей машины с помощью сетевой утилиты ipconfig
2. Добавьте IP-адрес в пропущенное поле конфигурации
3. Сохраните изменения в файле
4. Запустите программу-станцию из командной строки: python3 station.py --config configs/pk7.json

Для связи с Мастером и получения подсказок используйте канал спец. связи через открытое окно
на вашей Станции.

## Команды

```
send <текст>   — отправить сообщение соседу
inbox [n]      — показать последние n полученных сообщений
```
