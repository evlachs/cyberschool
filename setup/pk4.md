# Станция ПК4 — impostor («Крот»)

## Легенда

Вы — крот, агент противника, внедрённый в самое сердце канала связи, между двух архивных
станций, способных разоблачить вас, если сверят свои записи друг с другом. Ваша задача двойная:
перехватывать всё, что проходит через вас, вскрывая шифры, которые команда считает надёжной
защитой, и, при удачном случае, подменять содержимое сообщения прежде, чем оно уйдёт дальше.

## Ваша задача

Читайте всё, что проходит через вас (`list`, `show`), и вскрывайте шифры (`crack_caesar`,
`crack_xor`), чтобы понимать, что на самом деле происходит в канале. Решайте, когда подменять:
можно включить ручной режим (`manual on`) и рассматривать каждое сообщение перед тем, как отпустить его (`release`) —
в исходном виде или с подменой. Помните: обе базы данных способны сравнить свои записи и вычислить
любую вашу подмену — действуйте точно и выбирайте момент с умом.

## Запуск

Запустить программу-станцию:
1. Узнайте IP-адрес вашей машины с помощью сетевой утилиты ipconfig
2. Добавьте IP-адрес в пропущенное поле конфигурации
3. Сохраните изменения в файле
4. Запустите программу-станцию из командной строки: python3 station.py --config configs/pk4.json

Для связи с Мастером и получения подсказок используйте канал спец. связи через открытое окно
на вашей Станции.

## Команды

```
list [n]                 — перехваченные сообщения (обе стороны)
show <msg_id>             — показать сообщение целиком
crack_caesar <id> [сдвиг] — расшифровать конкретным сдвигом либо перебрать все 1..32
crack_xor <id> <ключ>     — попробовать расшифровать XOR предполагаемым ключом

manual [on|off]           — ручной режим: входящие НЕ пересылаются сами, ждут вашего release
pending                   — сообщения, которые сейчас ждут вашего решения (ручной режим)
release <id> [текст]      — отпустить конкретное: без текста — как пришло, с текстом — подмена
release_all               — отпустить всё разом без изменений (если накопилось и надо разгрузить)
```
